EPIC 251 — Geophysical Computing

An introduction to computer programming, typically for sophomores majoring in Geophysics at Mines.

#content()

Summary

Geophysicists today use some of the most powerful computers in the world. This is partly because geophysical data volumes are vast and growing. Another reason is that raw geophysical data are not easily understood, and typically require substantial processing and interactive visualization and exploration before we can make sense of them.

This section of EPICS 251 introduces students of geophysics at Mines to the development of computer software for analyzing geophysical data. We will use freely available tools, such as the programming languages Python and Java, and free libraries of software written in these and other languages. We will work in teams, and practice good software engineering.

We will analyze freely available geophysical data. The image shown here is an example of data available from the National Geophysical Data Center . Can you guess what it is?

Although this section of EPICS 251 is designed for geophysics majors, others (particularly those with a strong interest in computing) are welcome with permission of the instructor.

Resources

The computers in GC 297 run Centos 5, a GNU/Linux operating system based on Red Hat Enterprise Linux 5. These systems include open-source software that you can freely download and install on your own computers. All of the software tools used in this course work well on systems running GNU/Linux, Microsoft Windows, and Mac OS X operating systems.

You can download and install Python on your computer. (If you use Linux or Mac OS X, you already have Python, but may want a more recent version. Currently we use version 2.4.)

For scientific computing, we use NumPy and SciPy. The latter depends on the former.

For 2-D graphics and plotting, we use Matplotlib (pylab).

Jython is Python for Java programmers.
